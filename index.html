О приложении

Art-Lab1 — это личное приложение для автоматической загрузки изображений (пинов) в Pinterest через официальный Pinterest API. Программа используется владельцем для собственных публикаций.

Возможности

  • Публикация изображений в выбранные доски Pinterest
  • Автоматическое добавление заголовков и описаний
  • Поддержка ссылок на товары и страницы
  • Безопасная работа через официальный API Pinterest
