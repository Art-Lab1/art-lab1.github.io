Политика конфиденциальности Art-Lab1

Последнее обновление: октябрь 2025

Эта Политика описывает, как Art-Lab1 обрабатывает данные при загрузке изображений (пинов) в Pinterest через официальный API Pinterest.

1. Обрабатываемые данные

  • OAuth-токен Pinterest, срок действия и ID доски — хранятся локально у владельца.
  • Контент для публикации: изображения, заголовки, описания и ссылки, загружаемые владельцем в свой Pinterest-аккаунт.
  • Локальные журналы ошибок (при необходимости): сообщения об ошибках, временные метки. Не передаются третьим лицам.

2. Использование данных

Данные применяются исключительно для авторизации и публикации контента владельцем через API Pinterest.

3. Хранение и безопасность

  • Токены и настройки хранятся локально и не публикуются.
  • Опубликованный контент хранится в Pinterest и регулируется их политиками.

4. Передача данных

Данные не передаются третьим лицам, кроме Pinterest в рамках API-взаимодействия. Продажа или аренда данных не осуществляется.

5. Отзыв доступа и удаление

  • OAuth-доступ можно отозвать в настройках Pinterest.
  • Удалить пины можно в интерфейсе Pinterest.
  • Локальные журналы и токены можно удалить вручную.

6. Изменения политики

Политика может обновляться, актуальная версия доступна по этому адресу.
